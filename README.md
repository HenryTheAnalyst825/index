# 🐾 PetFud Project - Pet Care Interface (Flat Structure Optimization)

A streamlined web interface project designed for pet services. This repository demonstrates core front-end development and version control using a **Flat Directory Structure** for maximum path reliability and rapid deployment.

## 🚀 Live Demo
Access the live site here: [https://henrytheanalyst825.github.io/index/](https://henrytheanalyst825.github.io/index/)

## 🛠 Tech Stack
* **HTML5 & CSS3**: Core architecture and custom pet-themed styling.
* **JavaScript**: Interactive elements and dynamic behavior (`Pet.js`).
* **Bootstrap 5**: Responsive layout ensuring mobile-first compatibility.
* **GitHub Desktop**: Agile version control and deployment.

## 📂 Project Management Logic (Architecture)
In this project, I applied a **Lean Management** approach by using a **Flat Directory Structure**:
* **Efficiency**: All `.html` files (index, products, cart, etc.) are located in the root directory to eliminate path errors and ensure 100% link integrity.
* **Asset Organization**: Images are centralized in the `imagespetfud/` folder for clean resource management.
* **Scalability**: Designed for quick updates without the complexity of nested sub-directories.
